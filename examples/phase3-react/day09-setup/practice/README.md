# Day 09 연습문제 - Vite, JSX, 첫 컴포넌트

## 학습 목표
- Vite로 React 프로젝트를 생성하고 실행할 수 있다
- JSX 문법으로 UI를 작성할 수 있다
- 변수 바인딩(`{}`)과 배열 렌더링(`map`)을 활용할 수 있다

---

## 문제 1: 프로필 카드

자신의 이름, 직업, 취미 3개를 보여주는 **프로필 카드** 컴포넌트를 만드세요.

### 요구사항
- `name`, `job`, `hobbies` 변수를 선언하고 JSX에서 `{}`로 바인딩하세요
- `hobbies`는 문자열 배열로 만들고, `map()`으로 `<li>` 태그를 렌더링하세요
- 카드 형태의 스타일을 인라인 스타일로 적용하세요 (border, padding, borderRadius 등)
- 프로필 이미지 대신 이름의 첫 글자를 큰 원 안에 표시하세요

### 예시 출력
```
  [홍]
  홍길동
  프론트엔드 개발자

  취미:
  • 코딩
  • 독서
  • 등산
```

## 문제 2: 메뉴판

음식점 메뉴판을 JSX로 렌더링하세요.

### 요구사항
- 음식 이름과 가격이 담긴 객체 배열을 만드세요 (최소 6개 메뉴)
- `map()`으로 메뉴 목록을 렌더링하세요
- 가격이 10,000원 이상이면 **"인기"** 뱃지를 빨간색으로 표시하세요
- 가격은 `toLocaleString()`으로 천 단위 콤마를 표시하세요
- 전체 메뉴 개수를 상단에 표시하세요

### 예시 데이터
```typescript
const menuItems = [
  { id: 1, name: '김치찌개', price: 8000 },
  { id: 2, name: '한우 불고기', price: 15000 },
  // ...
];
```

## 문제 3: 시간표

요일별 수업 시간표를 테이블 형태로 렌더링하세요.

### 요구사항
- 요일(월~금)과 교시(1~4교시)를 포함하는 2차원 데이터를 만드세요
- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>` 태그를 사용하세요
- `map()`을 중첩하여 행과 열을 동적으로 렌더링하세요
- 빈 시간(없는 수업)은 "-"로 표시하세요
- 테이블에 기본 스타일을 적용하세요 (border, padding 등)

### 예시 데이터
```typescript
const days = ['월', '화', '수', '목', '금'];
const timetable = [
  ['국어', '수학', '영어', '과학', '체육'],   // 1교시
  ['수학', '국어', '사회', '영어', '음악'],   // 2교시
  // ...
];
```

## 실행 방법

```bash
# 프로젝트 생성 (이미 생성된 경우 생략)
npm create vite@latest practice -- --template react-ts
cd practice
npm install

# App.tsx 파일을 수정한 후
npm run dev
```

## 힌트
- JSX에서 `map()` 사용 시 반드시 `key` prop을 지정하세요
- 인라인 스타일은 `style={{ color: 'red' }}` 형태로 작성합니다
- 조건부 렌더링: `{조건 && <태그>내용</태그>}` 또는 삼항 연산자 사용
