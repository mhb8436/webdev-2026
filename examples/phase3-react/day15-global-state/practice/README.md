# Day 15 - Context API 연습문제

Context API를 활용하여 전역 상태를 관리하는 3가지 연습문제입니다.
각 문제는 독립적으로 풀 수 있으며, `App.tsx`의 스켈레톤 코드를 완성하세요.

---

## 문제 1: 다크모드 토글

### 요구사항

- `ThemeContext`를 생성하여 앱 전체의 다크/라이트 모드를 관리하세요
- `ThemeProvider` 컴포넌트를 만들어 테마 상태와 토글 함수를 제공하세요
- `useTheme` 커스텀 훅을 만들어 어느 컴포넌트에서든 테마에 접근할 수 있게 하세요
- `Header`, `Content`, `Footer` 3개 컴포넌트 모두에 테마를 적용하세요
- 다크모드일 때 배경은 `#1a1a2e`, 글자색은 `#e0e0e0`
- 라이트모드일 때 배경은 `#ffffff`, 글자색은 `#333333`

### 예시 출력

```
[라이트 모드 상태]
┌──────────────────────────┐
│ Header    [다크모드 전환]  │  ← 흰 배경, 어두운 글자
├──────────────────────────┤
│                          │
│  안녕하세요!              │  ← 흰 배경, 어두운 글자
│  현재 테마: 라이트 모드    │
│                          │
├──────────────────────────┤
│ Footer - 라이트 모드      │  ← 흰 배경, 어두운 글자
└──────────────────────────┘

[버튼 클릭 후 - 다크 모드 상태]
┌──────────────────────────┐
│ Header    [라이트모드 전환] │  ← 어두운 배경, 밝은 글자
├──────────────────────────┤
│                          │
│  안녕하세요!              │  ← 어두운 배경, 밝은 글자
│  현재 테마: 다크 모드      │
│                          │
├──────────────────────────┤
│ Footer - 다크 모드        │  ← 어두운 배경, 밝은 글자
└──────────────────────────┘
```

### 힌트

- `createContext`로 `ThemeContext`를 만들고 초기값은 `null`로 설정하세요
- `ThemeProvider` 안에서 `useState`로 `isDark` 상태를 관리하세요
- `useTheme` 훅에서 `useContext(ThemeContext)`를 호출하고, 값이 없으면 에러를 던지세요
- 각 컴포넌트의 `style` 속성에 `isDark` 값에 따라 다른 색상을 적용하세요

## 문제 2: 다국어 지원 (i18n)

### 요구사항

- `LanguageContext`를 생성하여 한국어/영어 전환 기능을 구현하세요
- `LanguageProvider` 컴포넌트를 만들어 현재 언어 상태와 전환 함수를 제공하세요
- `useLanguage` 커스텀 훅을 만들어 현재 언어, 전환 함수, 번역 함수(`t`)를 반환하세요
- 번역 데이터 객체(`translations`)를 만들어 키 기반으로 번역을 관리하세요
- 최소 3개 컴포넌트(`NavigationBar`, `MainContent`, `ProfileCard`)에서 번역이 동작하게 하세요
- 번역 키에 해당하는 값이 없으면 키 자체를 반환하세요

### 예시 출력

```
[한국어 모드]
┌──────────────────────────────┐
│ 홈  소개  연락처    [English] │
├──────────────────────────────┤
│                              │
│  다국어 지원 앱에 오신 것을    │
│  환영합니다!                  │
│                              │
│  ┌─────────────┐             │
│  │ 사용자 프로필 │             │
│  │ 이름: 홍길동  │             │
│  │ 역할: 개발자  │             │
│  └─────────────┘             │
│                              │
└──────────────────────────────┘

[English 버튼 클릭 후]
┌──────────────────────────────┐
│ Home  About  Contact  [한국어]│
├──────────────────────────────┤
│                              │
│  Welcome to the              │
│  Multilingual App!           │
│                              │
│  ┌──────────────┐            │
│  │ User Profile  │            │
│  │ Name: Hong    │            │
│  │ Role: Developer│           │
│  └──────────────┘            │
│                              │
└──────────────────────────────┘
```

### 힌트

- 번역 데이터 구조 예시:
  ```typescript
  const translations = {
    ko: {
      'nav.home': '홈',
      'nav.about': '소개',
      'nav.contact': '연락처',
      'welcome': '다국어 지원 앱에 오신 것을 환영합니다!',
      ...
    },
    en: {
      'nav.home': 'Home',
      'nav.about': 'About',
      'nav.contact': 'Contact',
      'welcome': 'Welcome to the Multilingual App!',
      ...
    }
  };
  ```
- `t` 함수는 `(key: string) => string` 형태로, 현재 언어에 해당하는 번역 값을 반환합니다
- `useLanguage` 훅이 `{ language, setLanguage, t }` 객체를 반환하도록 설계하세요

## 문제 3: 장바구니 전역 상태

### 요구사항

- `CartContext`를 생성하여 장바구니를 전역으로 관리하세요
- `CartProvider` 컴포넌트를 만들어 장바구니 상태와 관련 함수들을 제공하세요
- `useCart` 커스텀 훅을 만들어 장바구니 기능에 접근할 수 있게 하세요
- 제공해야 할 기능: `addToCart`, `removeFromCart`, `updateQuantity`, `getTotalCount`, `getTotalPrice`
- `ProductList` 컴포넌트: 상품 목록을 표시하고 장바구니에 추가하는 버튼 제공
- `Cart` 컴포넌트: 장바구니 목록을 표시하고 수량 변경/삭제 기능 제공
- `CartIcon` 컴포넌트: 현재 장바구니에 담긴 총 상품 개수를 배지로 표시

### 예시 출력

```
┌──────────────────────────────────┐
│ 쇼핑몰                    🛒 (3) │  ← CartIcon: 총 개수 배지
├──────────────────────────────────┤
│                                  │
│  [상품 목록]                      │
│  ┌────────────────────────────┐  │
│  │ 노트북    1,200,000원       │  │
│  │               [장바구니 추가]│  │
│  ├────────────────────────────┤  │
│  │ 키보드      89,000원        │  │
│  │               [장바구니 추가]│  │
│  ├────────────────────────────┤  │
│  │ 마우스      45,000원        │  │
│  │               [장바구니 추가]│  │
│  └────────────────────────────┘  │
│                                  │
│  [장바구니]                       │
│  ┌────────────────────────────┐  │
│  │ 노트북 [-] 2 [+] [삭제]    │  │
│  │ 키보드 [-] 1 [+] [삭제]    │  │
│  ├────────────────────────────┤  │
│  │ 총 금액: 2,489,000원       │  │
│  └────────────────────────────┘  │
│                                  │
└──────────────────────────────────┘
```

### 힌트

- 장바구니 아이템 타입:
  ```typescript
  interface CartItem {
    id: number;
    name: string;
    price: number;
    quantity: number;
  }
  ```
- 상품 데이터는 컴포넌트 밖에 상수로 정의하세요
- `addToCart` 시 이미 있는 상품이면 수량만 증가시키세요
- `updateQuantity`에서 수량이 0 이하가 되면 자동으로 삭제하세요
- 가격 표시에 `toLocaleString()`을 사용하면 천 단위 구분이 됩니다

## 공통 참고사항

- 모든 문제는 `App.tsx` 한 파일에서 작업합니다
- 정답은 `solution/App.tsx`에서 확인할 수 있습니다
- Context를 사용할 때는 반드시 Provider 안에서 사용해야 합니다
- 커스텀 훅을 만들면 코드가 깔끔해지고 재사용성이 높아집니다
- TypeScript 타입을 정의하면 자동완성과 에러 방지에 도움이 됩니다
