# Day 12 연습문제 - 조건부 렌더링, 필터

## 학습 목표
- 조건부 렌더링으로 상황에 따라 다른 UI를 표시할 수 있다
- 필터링을 통해 데이터를 선택적으로 표시할 수 있다
- 조건부 스타일링을 적용할 수 있다

---

## 문제 1: 학생 성적표

학생 목록을 필터링하고 조건부 스타일링을 적용하세요.

### 요구사항
- 학생 데이터(이름, 과목, 점수) 배열을 준비하세요 (최소 8명)
- "전체", "합격", "불합격" 필터 버튼 3개를 만드세요
- 60점 이상이면 합격, 미만이면 불합격입니다
- 합격자는 초록색 배경, 불합격자는 빨간색 배경으로 조건부 스타일링하세요
- 선택된 필터 버튼은 하이라이트하세요
- 필터링된 학생 수를 표시하세요
- 평균 점수도 함께 표시하세요

### 예시 데이터
```typescript
const students = [
  { id: 1, name: '김철수', subject: '수학', score: 85 },
  { id: 2, name: '이영희', subject: '수학', score: 45 },
  // ...
];
```

## 문제 2: 날씨 앱 UI

도시별 날씨 정보를 조건부 렌더링하세요.

### 요구사항
- 도시별 날씨 데이터를 준비하세요 (최소 5개 도시)
- 날씨 상태: 맑음, 흐림, 비, 눈 4가지
- 각 날씨에 따라 다른 이모지를 표시하세요 (맑음: ☀️, 흐림: ☁️, 비: 🌧️, 눈: ❄️)
- 각 날씨에 따라 다른 배경색을 적용하세요
- 도시 목록을 버튼으로 만들고, 클릭하면 해당 도시의 날씨를 상세히 표시하세요
- 온도, 습도 등 추가 정보도 표시하세요
- 선택된 도시 버튼은 하이라이트하세요

### 예시 데이터
```typescript
type Weather = '맑음' | '흐림' | '비' | '눈';

const cities = [
  { name: '서울', weather: '맑음' as Weather, temp: 22, humidity: 45 },
  { name: '부산', weather: '흐림' as Weather, temp: 19, humidity: 60 },
  // ...
];
```

## 문제 3: 탭 컴포넌트

탭 전환으로 다른 내용을 조건부 렌더링하세요.

### 요구사항
- "소개", "경력", "프로젝트" 3개 탭을 만드세요
- 선택된 탭에 따라 다른 내용을 조건부 렌더링하세요
- 선택된 탭은 하이라이트 스타일을 적용하세요 (배경색, 글자색, 밑줄 등)
- 각 탭의 내용은 자유롭게 작성하되, 최소한의 텍스트와 리스트를 포함하세요
- 탭 전환 시 부드러운 시각적 피드백을 주세요

### 탭 내용 예시
- **소개**: 이름, 자기소개, 관심 분야 등
- **경력**: 회사명, 기간, 직무 등의 리스트
- **프로젝트**: 프로젝트명, 설명, 사용 기술 등의 리스트

## 힌트
- 조건부 렌더링: `{condition && <Component />}` 또는 삼항 연산자
- 필터링: `array.filter(item => item.score >= 60)`
- 조건부 스타일: `style={{ color: isActive ? 'blue' : 'gray' }}`
- 조건부 className: `className={isActive ? 'active' : ''}`
