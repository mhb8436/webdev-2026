# Day 01 연습 문제 - 변수, 자료형, 배열, console.log

> 메인 실습과는 다른 시나리오로 같은 개념을 연습합니다.

## 문제 1: 자기소개 프로그램

이름, 나이, 직업을 각각 변수에 저장하고, 한 줄씩 `console.log`로 출력하세요.
`typeof` 연산자를 사용하여 각 변수의 타입도 함께 출력하세요.

**예상 출력:**
```
이름: 김민수
나이: 25
직업: 개발자
이름의 타입: string
나이의 타입: number
직업의 타입: string
```

## 문제 2: 쇼핑 목록

사고 싶은 물건 5개를 배열에 저장하고, 번호를 붙여서 출력하세요.
총 몇 개의 물건이 있는지도 출력하세요. (`length` 속성 활용)

**예상 출력:**
```
=== 쇼핑 목록 ===
1. 노트북
2. 키보드
3. 마우스
4. 모니터
5. 헤드셋
총 5개의 물건이 있습니다.
```

## 문제 3: 온도 변환기

섭씨 온도를 변수에 저장하고, 화씨로 변환하여 출력하세요.

**변환 공식:** `화씨 = 섭씨 * 9/5 + 32`

여러 온도(0도, 25도, 37도, 100도)에 대해 변환 결과를 출력하세요.

**예상 출력:**
```
섭씨 0도 → 화씨 32도
섭씨 25도 → 화씨 77도
섭씨 37도 → 화씨 98.6도
섭씨 100도 → 화씨 212도
```

## 문제 4: 좋아하는 것 목록

좋아하는 음식, 영화, 노래를 각각 배열로 만드세요 (각 3개 이상).
각 배열의 첫 번째 항목과 배열의 길이(총 개수)를 출력하세요.

**예상 출력:**
```
=== 좋아하는 것 목록 ===
[음식] 총 3개 - 가장 좋아하는 것: 피자
[영화] 총 3개 - 가장 좋아하는 것: 인터스텔라
[노래] 총 3개 - 가장 좋아하는 것: Dynamite
```

## 힌트

- 변수 선언: `let`, `const` 사용
- 배열 선언: `const arr = [항목1, 항목2, ...]`
- 배열 길이: `arr.length`
- 배열 인덱스: `arr[0]` (첫 번째), `arr[1]` (두 번째)
- 타입 확인: `typeof 변수명`
- 문자열 연결: 템플릿 리터럴 `` `${변수}` `` 또는 `+` 연산자
