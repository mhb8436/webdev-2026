## 문제 1: 간단한 카운터

`+`, `-` 버튼과 현재 숫자를 표시하는 카운터를 만드세요.

**요구사항:**
- `+` 버튼 클릭 시 숫자가 1 증가
- `-` 버튼 클릭 시 숫자가 1 감소
- `리셋` 버튼 클릭 시 0으로 초기화
- **0 미만으로 내려가지 않게** 하세요 (최소값 0)
- 현재 숫자를 화면에 크게 표시하세요

## 문제 2: 실시간 글자수 세기

`textarea`에 글을 입력하면 실시간으로 글자수와 단어수를 표시하세요.

**요구사항:**
- 입력할 때마다 글자수와 단어수를 실시간 업데이트
- 단어수는 공백으로 구분 (빈 입력이면 0)
- **100자 이상이면 글자수를 빨간색으로** 경고 표시
- 100자 미만이면 다시 정상 색상으로 복원

---

## 문제 3: 랜덤 색상 생성기

버튼을 클릭하면 배경색이 랜덤으로 바뀌는 프로그램을 만드세요.

**요구사항:**
- `색상 변경` 버튼 클릭 시 배경색 랜덤 변경
- 현재 색상 코드(`#RRGGBB`)를 화면에 표시
- **색상 히스토리** 표시 (최근 5개)
- 히스토리의 각 항목은 해당 색상으로 배경 표시

## 문제 4: 간단한 퀴즈 앱

3개의 객관식 퀴즈 문제를 만들고, 사용자가 답을 선택하면 결과를 보여주세요.

**요구사항:**
- 3개의 퀴즈 문제 (각 4개의 보기)
- 보기 클릭 시 정답/오답 즉시 표시
- 정답이면 초록색, 오답이면 빨간색으로 표시
- **한 번 답을 선택하면 해당 문제는 변경 불가**
- 모든 문제를 풀면 총 점수(`N/3`)를 보여주세요

## 힌트

- 요소 선택: `document.getElementById('id')`, `document.querySelector('.class')`
- 이벤트 등록: `element.addEventListener('click', 함수)`
- 내용 변경: `element.textContent = '텍스트'`
- HTML 변경: `element.innerHTML = '<태그>내용</태그>'`
- 스타일 변경: `element.style.color = 'red'`, `element.style.backgroundColor = '#fff'`
- 클래스 추가/제거: `element.classList.add('class')`, `element.classList.remove('class')`
- 랜덤 색상: `'#' + Math.floor(Math.random() * 16777215).toString(16).padStart(6, '0')`
