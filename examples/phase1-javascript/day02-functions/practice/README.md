## 문제 1: 계산기 함수

`add`, `subtract`, `multiply`, `divide` 4개의 함수를 만들어 테스트하세요.
각 함수는 두 개의 숫자를 받아 계산 결과를 반환합니다.
`divide` 함수는 0으로 나누려고 하면 `"0으로 나눌 수 없습니다"`를 반환하세요.

**예상 출력:**
```
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3.3333333333333335
10 / 0 = 0으로 나눌 수 없습니다
```

## 문제 2: 성적 판별기

점수(0~100)를 받아 학점을 반환하는 `getGrade` 함수를 만드세요.

**학점 기준:**
- 90점 이상: A
- 80점 이상: B
- 70점 이상: C
- 60점 이상: D
- 60점 미만: F

여러 점수에 대해 테스트하세요 (예: 95, 82, 73, 61, 45).

**예상 출력:**
```
95점 → A
82점 → B
73점 → C
61점 → D
45점 → F
```

## 문제 3: 구구단 출력기

숫자를 받아 해당 단의 구구단을 출력하는 `printMultiTable` 함수를 만드세요.
반복문(`for`)을 사용하여 1부터 9까지 곱하세요.

**예상 출력 (3단의 경우):**
```
=== 3단 ===
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
...
3 x 9 = 27
```

## 문제 4: 배열 통계

숫자 배열을 받아 합계, 평균, 최대값, 최소값을 계산하여 반환하는 `getStats` 함수를 만드세요.
결과는 객체 `{ sum, average, max, min }` 형태로 반환하세요.

**예상 출력:**
```
입력: [10, 20, 30, 40, 50]
합계: 150
평균: 30
최대값: 50
최소값: 10
```

## 힌트

- 함수 선언: `function 함수이름(매개변수) { ... return 결과; }`
- 조건문: `if`, `else if`, `else`
- 반복문: `for (let i = 0; i < 길이; i++) { ... }`
- 배열 순회: `for (let i = 0; i < arr.length; i++)`
- `Math.max()`, `Math.min()` 또는 반복문으로 최대/최소 구하기
