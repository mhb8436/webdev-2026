## 힌트

- 객체 배열: `const arr = [{ key: value }, { key: value }]`
- `filter`: `arr.filter(item => 조건)`  -- 조건에 맞는 요소만 모은 새 배열
- `map`: `arr.map(item => 변환)` -- 각 요소를 변환한 새 배열
- `find`: `arr.find(item => 조건)` -- 조건에 맞는 첫 번째 요소
- `sort`: `arr.sort((a, b) => b.점수 - a.점수)` -- 내림차순 정렬
- `forEach`: `arr.forEach(item => { ... })` -- 각 요소에 대해 실행
- `Date`: `new Date()`, `new Date('2025-12-25')`
- 날짜 차이 (밀리초): `날짜2 - 날짜1`, 일수 변환: `/ (1000 * 60 * 60 * 24)`
