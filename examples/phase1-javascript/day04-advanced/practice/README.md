## 문제 1: 객체 병합기

두 객체를 spread 연산자(`...`)로 병합하는 `mergeObjects` 함수를 만드세요.
중복된 키가 있으면 두 번째 객체의 값이 우선합니다.

**요구사항:**
- 화살표 함수로 구현하세요.
- 원본 객체를 변경하지 않아야 합니다.
- 병합 결과를 새로운 객체로 반환하세요.

**예상 출력:**
```
원본1: { name: '김철수', age: 25, city: '서울' }
원본2: { age: 30, job: '개발자', city: '부산' }
병합: { name: '김철수', age: 30, city: '부산', job: '개발자' }
```

## 문제 2: 배열 유틸리티

화살표 함수로 다음 3가지 유틸리티 함수를 구현하세요:

1. **`unique(arr)`** - 배열에서 중복을 제거하세요. (`Set` 활용)
2. **`shuffle(arr)`** - 배열을 랜덤으로 섞으세요. (원본 변경 금지)
3. **`chunk(arr, size)`** - 배열을 지정된 크기로 나누세요.

**예상 출력:**
```
원본: [1, 2, 2, 3, 3, 3, 4, 5]
중복 제거: [1, 2, 3, 4, 5]

원본: [1, 2, 3, 4, 5]
섞기: [3, 1, 5, 2, 4]  (매번 다름)

원본: [1, 2, 3, 4, 5, 6, 7]
2개씩 나누기: [[1,2], [3,4], [5,6], [7]]
3개씩 나누기: [[1,2,3], [4,5,6], [7]]
```

## 문제 3: 구조분해 연습

다음 중첩된 객체에서 구조분해 할당을 사용하여 값을 추출하세요:

```javascript
const data = {
    user: {
        name: '이영희',
        address: {
            city: '서울',
            zip: '06123'
        }
    },
    scores: [90, 85, 78]
};
```

**추출할 값:**
1. `city` (주소에서)
2. `zip` (주소에서)
3. 첫 번째 점수를 `firstScore`로
4. 나머지 점수를 `restScores`로
5. `name`을 `userName`으로 이름 변경

**예상 출력:**
```
도시: 서울
우편번호: 06123
첫 번째 점수: 90
나머지 점수: [85, 78]
사용자 이름: 이영희
```

## 문제 4: reduce 마스터

`reduce`를 사용하여 다음 3가지를 구현하세요:

1. **배열의 최대값 찾기**: `Math.max` 없이 reduce만으로 구현
2. **문자열 배열을 하나의 문장으로 합치기**: `['나는', '자바스크립트를', '공부합니다']` -> `'나는 자바스크립트를 공부합니다'`
3. **배열의 요소 빈도수 카운팅**: `['a', 'b', 'a', 'c', 'b', 'a']` -> `{ a: 3, b: 2, c: 1 }`

**예상 출력:**
```
배열: [5, 3, 9, 1, 7, 2, 8]
최대값: 9

단어: ['나는', '자바스크립트를', '공부합니다']
문장: 나는 자바스크립트를 공부합니다

배열: ['a', 'b', 'a', 'c', 'b', 'a']
빈도수: { a: 3, b: 2, c: 1 }
```

## 힌트

- 화살표 함수: `const fn = (param) => { ... }` 또는 `const fn = (param) => 한줄표현식`
- Spread 연산자: `{ ...obj1, ...obj2 }`, `[...arr1, ...arr2]`
- 구조분해: `const { key1, key2 } = obj`, `const [a, b, ...rest] = arr`
- 중첩 구조분해: `const { outer: { inner } } = obj`
- 이름 변경: `const { name: userName } = obj`
- `Set`으로 중복 제거: `[...new Set(arr)]`
- `reduce`: `arr.reduce((누적값, 현재값) => { ... }, 초기값)`
